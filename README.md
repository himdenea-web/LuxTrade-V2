##  Présentation
LuxTrade est une application web éducative conçue pour initier les utilisateurs au monde du trading. L'application permet de suivre les cours réels des crypto-monnaies et de s'exercer à l'achat/revente avec un solde virtuel.

##  Stack Technique

Frontend : HTML5 / CSS3 .

Langage : JavaScript ES6 .

Données : API externe CoinGecko pour des prix certifiés en temps réel.

##  Fonctionnalités Clés

Sélecteur Multi-Devises : Passage instantané entre Bitcoin, Ethereum et Solana.

Live Pricing : Actualisation automatique des cours toutes les 15 secondes.

Gestion de Portefeuille : Calcul automatique des quantités détenues en fonction du cours au moment de l'achat.

Journal de Bord : Historique complet des transactions pour analyser ses performances.

##  Objectifs Pédagogiques

Maîtriser les appels API asynchrones.

Gérer des structures de données complexes .

Créer une interface utilisateur  haut de gamme répondant aux standards du secteur financier luxembourgeois.
